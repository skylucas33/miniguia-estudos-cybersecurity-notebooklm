## 🤖 Testes de Prompts no NotebookLM

Prompt 1:
"O que é Cybersecurity?"

Resposta resumida:
"Cibersegurança é definida como o conjunto de tecnologias, processos e práticas projetados para proteger redes, dispositivos, programas e dados contra ataques, danos ou acessos não autorizados."

Prompt 2:
"Explique phishing com exemplos."

Resultado:
O phishing é um ataque de engenharia social que utiliza comunicações fraudulentas (como e-mails, SMS e chamadas telefônicas) para manipular as pessoas a compartilharem dados confidenciais, baixarem malwares ou realizarem transferências financeiras indevidas

Phishing em massa: Mensagens genéricas enviadas para milhões de pessoas, como um e-mail falso de uma marca global (ex: Amazon ou Microsoft) alegando um "problema com seu pedido" ou uma "fatura vencida em anexo"

Spear Phishing: Um ataque altamente direcionado a um indivíduo ou grupo específico, utilizando pesquisas detalhadas (muitas vezes via LinkedIn) para citar nomes reais de colegas ou fornecedores, tornando a mensagem muito convincente.


Dificuldade encontrada:
Algumas respostas ficaram muito genéricas e precisei pedir exemplos práticos.


## 📖 Miniguia de Estudo

### Fundamentos da Cybersecurity
Os fundamentos da cibersegurança baseiam-se na proteção de sistemas, redes e dados contra ataques digitais através de uma abordagem que integra tecnologias, processos e pessoas

## Principais Tipos de Ataques
- Phishing
- Malware
- Ransomware

### Boas Práticas de Segurança
- Utilizar autenticação de dois fatores
- Atualizar sistemas regularmente
- Evitar links suspeitos
- 

## 📚 Glossário

Cybersecurity → proteção de sistemas e dados contra ataques digitais.

Phishing → tentativa de enganar usuários para roubar informações.

Malware → software malicioso.
